Git

設定.gitignore_global

git config –global core.excludesfile ~/.gitignore_global

將現有專案push上gihub.com

  1. 於github.com建立repository

  2. 於專案目錄下:

    1. git init
    2. git add -A
    3. git commit -m “init”
    4. git remote add origin {github.com上repository的HTTPS clone URL}

    5. git push -u origin master

    6. git status

  3. github.com的repository有檔案，Done.
